Detailed Description

Generic AStar implementation Designed to be subclassed to provide specific node expansion and costing.

CostNode() and Expand() must be overridden.

To perform searches the Search() method is passed a start node and an end node. It returns null if a path is impossible otherwise a list of nodes. Node values can be any object.

You can optionally override Search(Object start, Object end) and IsEndNode(AStarNode node) but the default implementations are functional.

◆ cost

virtual float StompyBlondie.Math.AStar.cost

Should be overidden by subclasses to return the cost of a node, indicating to the algorithm how preferable the choice of move is.

Parameters
nodeThe node to cost up.
Returns
A tuple containing cost and pathCost values respectively, or null if the node should not be considered at all (an impossible move). pathCost is the total cost to move along the path from the start to this node, and cost is pathCost plus an estimate of the distance to the goal. The pathCost can be found by adding the movement cost of this node to node.previous.pathCost, if available.
